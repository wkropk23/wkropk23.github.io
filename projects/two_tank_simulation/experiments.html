Instructions

This page demonstrates how different controller settings and system parameters affect the two-tank system dynamics. Each experiment includes a plot or screenshot and key observations to aid learning.

For interactive simulation, please visit the live simulation page.

Proportional Control Only

Observation: The tanks respond quickly to disturbances, but steady-state error remains. This demonstrates why integral action is needed.

Integral Control Only

Observation: Steady-state error is eliminated, but response may be slower and overshoot can occur.

High Proportional Gain (Kp)

Observation: Overshoot is significant and oscillations may appear. Illustrates the risk of too high Kp.

Proportional + Integral Control

Observation: Steady-state error is removed and response is faster. However, if Ki is too high, oscillations or instability can occur.

Effect of Inlet Pressure (P₁)

Observation: Increasing P₁ temporarily raises H₁ and H₂. The PID controllers compensate to bring the tanks back to the setpoint.

Summary

These experiments highlight key lessons about PID control, system dynamics, and how changes in system inputs affect interconnected tanks. Observing these responses helps in tuning controllers and understanding multi-variable interactions.

For hands-on exploration, use the interactive simulation.
