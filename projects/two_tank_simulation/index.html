Overview

This project models two interconnected liquid tanks with PID level control, designed to simulate fluid dynamics in chemical engineering processes. Users can adjust PID parameters and inlet flow rates to observe real-time system responses, achieving stable control under varying conditions. Built as a demonstration of process control, it integrates numerical modeling with an interactive web interface.

Features

  • Adjustable PID parameters (proportional, integral, derivative) in both independent and dependent forms for flexible control tuning.
  • Real-time visualization of tank levels using Chart.js, updating dynamically with user inputs.
  • Robust backend with Python, Flask for server-side logic, and SciPy for solving differential equations.
  • Responsive web interface built with HTML and CSS, optimized for desktop and mobile.
